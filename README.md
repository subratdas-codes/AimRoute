# AimRoute

AimRoute is a career guidance and roadmap platform that helps users discover, plan, and navigate their professional journey.

## Features

- 🎯 Personalized career roadmaps
- 🔍 Role and skill discovery
- 📈 Progress tracking
- 🗺️ Step-by-step learning paths

## Getting Started

### Prerequisites

- Node.js (v18 or above)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/subratdas-codes/AimRoute.git

# Navigate into the project directory
cd AimRoute

# Install dependencies
npm install
```

### Running the App

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
